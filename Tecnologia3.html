TECNOLOGÍA E INFORMÁTICA
Excel es una herramienta muy eficaz para obtener información con significado a partir de grandes cantidades de datos. También funciona muy bien con cálculos sencillos y para realizar el seguimiento de casi cualquier tipo de información. La clave para desbloquear todo este potencial es la cuadrícula de las celdas. ¿Cómo funciona Excel? Funciona como una hoja de cálculo compuesta por filas y columnas divididas por celdas, y en donde se pueden llevar a cabo diversas operaciones aritméticas, así como con números y texto. Es decir, sumar, restar, multiplicar, etc.
